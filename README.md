# Розв'язки Завдань з Дисципліни "Прикладна Теорія Графів"

Цей репозиторій містить реалізації алгоритмів та розв'язків завдань з курсу "Прикладна Теорія Графів" мовою Python.

## Структура Проекту

### Lab 1: Властивості Графів (lab1.py)
Реалізація класу Graph з методами для аналізу властивостей графів:
- **BFS (Breadth-First Search)**: пошук найкоротшого шляху між вершинами
- **Матриця відстаней**: побудова матриці довжин найкоротших шляхів між усіма парами вершин
- **Розрахунок характеристик графа**:
  - Радіус (R) - мінімальна ексцентричність вершин
  - Діаметр (D) - максимальна ексцентричність вершин
  - Центральні вершини - вершини з ексцентричністю рівною радіусу
  - Периферійні вершини - вершини з ексцентричністю рівною діаметру

### Lab 2: Розфарбування Графів та Покриття (lab2.py)
Реалізація алгоритмів для розв'язання класичних задач теорії графів:
- **Розфарбування вершин графа**: жадібний алгоритм з сортуванням вершин за ступенем
- **Мінімальне вершинне покриття**: перебір всіх можливих підмножин вершин
- **Мінімальне реберне покриття**: жадібний алгоритм вибору ребер

### Lab 4: Сильно Зв'язані Компоненти (lab4.py)
Реалізація алгоритму Косараджу для знаходження сильно зв'язаних компонент:
- DFS для визначення порядку завершення вершин
- Транспонування графа
- Повторний DFS на транспонованому графі для виділення компонент

### Lab 5: Максимальний Потік (Ford-Fulkerson та Capacity Scaling)
Дві реалізації алгоритму максимального потоку:

#### lab5_ford-fulkerson.py
Класичний алгоритм Форда-Фалкерсона з використанням DFS для знаходження збільшуючих шляхів.

#### lab5_scaling.py
Оптимізована версія з масштабуванням пропускних здатностей для покращення ефективності.

### Lab 6: Алгоритм Дініца (lab6.py)
Ефективна реалізація алгоритму Дініца для обчислення максимального потоку:
- BFS для побудови шарового графа
- DFS для знаходження блокуючих потоків у межах шарів

### Lab 7: Задача Комівояжера - Метод Гілок та Меж (lab7.py)
Реалізація методу гілок та меж для розв'язання задачі комівояжера:
- Перебір всіх можливих маршрутів з відсіканням за нижньою межею
- Використання матриці вартостей для оцінки вартості маршрутів

### Lab 8: Задача Комівояжера - Імітація Відпалу (lab8.py)
Реалізація метаевристичного алгоритму імітації відпалу для задачі комівояжера:
- Ітеративне покращення маршруту через випадкові перестановки
- Ймовірнісний прийом гірших рішень залежно від "температури"
- Поступове "охолодження" системи для збіжності до оптимуму

## Опис Алгоритмів

### Максимальний Потік
- **Ford-Fulkerson**: Знаходить збільшуючі шляхи за допомогою DFS
- **Capacity Scaling**: Оптимізує вибір шляхів, розглядаючи тільки ребра з достатньою пропускною здатністю
- **Dinic's Algorithm**: Використовує BFS для побудови шарів та DFS для блокуючих потоків

### Задача Комівояжера (TSP)
- **Branch and Bound**: Систематичний перебір з відсіканням неперспективних гілок
- **Simulated Annealing**: Метаевристика, що дозволяє тимчасово приймати гірші рішення
